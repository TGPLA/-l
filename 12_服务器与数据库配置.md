## 前端配置

### 配置文件位置
- 示例文件：`/.env.example`
- 实际配置：`/.env`（需自行创建）

### 配置项说明

| 配置项 | 说明 | 示例值 |
|--------|------|--------|
| `VITE_API_BASE_URL` | 后端 API 地址 | `http://localhost:8080/api` |
| `VITE_ZHIPU_API_KEY` | 智谱 AI API Key（可选） | 用户可在设置页面配置 |

### 本地开发
```bash
VITE_API_BASE_URL=http://localhost:8080/api
```

### 生产环境
```bash
VITE_API_BASE_URL=https://linyubo.top/api
```

## 后端配置

### 配置文件位置
- 示例文件：`/backend/.env.example`
- 实际配置：`/backend/.env`（需自行创建）

### 配置项说明

#### 服务配置
| 配置项 | 说明 | 示例值 |
|--------|------|--------|
| `SERVER_PORT` | 后端服务端口 | `8080` |

#### 数据库配置
| 配置项 | 说明 | 示例值 |
|--------|------|--------|
| `DB_HOST` | 数据库主机地址 | `127.0.0.1` 或 `1Panel-mysql-OHb5` |
| `DB_PORT` | 数据库端口 | `3306` |
| `DB_USER` | 数据库用户名 | `readrecall` |
| `DB_PASSWORD` | 数据库密码 | `your_password_here` |
| `DB_NAME` | 数据库名称 | `reading_reflection` |

#### JWT 配置
| 配置项 | 说明 | 示例值 |
|--------|------|--------|
| `JWT_SECRET` | JWT 签名密钥 | 随机生成的安全密钥 |

#### 智谱 AI 配置
| 配置项 | 说明 | 示例值 |
|--------|------|--------|
| `ZHIPU_API_KEY` | 智谱 AI API Key（可选） | 留空则使用用户配置 |
| `ZHIPU_MODEL` | 智谱 AI 模型 | `glm-4-flash` |

## 数据库连接方式

### 本地开发（SSH 隧道）

#### 第一步：建立 SSH 隧道
```bash
ssh -f -N -L 3307:127.0.0.1:3306 root@&lt;服务器IP&gt;
```

#### 第二步：配置后端 .env
```bash
DB_HOST=127.0.0.1
DB_PORT=3307
DB_USER=readrecall
DB_PASSWORD=&lt;数据库密码&gt;
DB_NAME=reading_reflection
```

### 服务器生产环境

#### Docker 容器网络
```bash
DB_HOST=1Panel-mysql-OHb5  # MySQL 容器名称
DB_PORT=3306
DB_USER=readrecall
DB_PASSWORD=&lt;数据库密码&gt;
DB_NAME=reading_reflection
```

## SSH 隧道配置

### Windows PowerShell
```powershell
# 建立隧道
ssh -f -N -L 3307:127.0.0.1:3306 root@linyubo.top

# 查看隧道进程
Get-Process | Where-Object {$_.ProcessName -eq "ssh"}

# 关闭隧道
Stop-Process -Name "ssh"
```

### macOS / Linux
```bash
# 建立隧道
ssh -f -N -L 3307:127.0.0.1:3306 root@linyubo.top

# 查看隧道进程
ps aux | grep ssh

# 关闭隧道
pkill -f "ssh -f -N -L 3307"
```

## 安全建议

1. **使用强密码**：数据库密码和 JWT 密钥应使用随机生成的强密码
2. **定期轮换**：定期更换数据库密码和 JWT 密钥
3. **最小权限**：数据库用户仅授予必要的权限
4. **不要提交**：永远不要将 `.env` 文件提交到 Git 仓库
5. **备份配置**：安全地备份生产环境的配置文件
