### 3.4 已知限制

**登录测试** 由于以下原因通过集成测试验证，而非 E2E：
- 注册成功后用户已登录，页面跳转到主页
- localStorage/Cookie 清除后页面状态不同步
- "立即登录"文本在登录状态下不显示

**解决方案**: 登录功能已通过 `local.integration.test.ts` 验证

## 五、调试指南

### 5.1 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| `ECONNREFUSED` | 后端服务未启动 | 启动后端: `cd backend && go run main.go` |
| `net::ERR_CONNECTION_REFUSED` | 前端服务未启动 | 启动前端: `npm run dev` |
| 500 状态码 | API 请求失败 | 检查后端日志 |
| 401 状态码 | 未授权访问 | 检查登录状态 |

### 5.2 调试步骤

```
1. 构建成功
2. 启动开发服务器
3. 打开浏览器
4. 检查控制台是否有错误
5. 定位并修复
```

### 5.3 查看测试报告

```bash
# Vitest 报告
npx vitest --reporter=verbose

# Playwright HTML 报告
npx playwright show-report
```
