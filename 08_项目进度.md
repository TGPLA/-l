### EPUB 导入弹窗文件选择器触发问题（2026-03-20）

**问题描述**：在 Chrome 浏览器中，点击 EPUB 导入弹窗的文件选择区域无法触发文件选择器。Trae 内置浏览器正常，Chrome 无反应，控制台无报错。

**已尝试但失败的方案（不要再尝试）**：
1. ❌ `<div onClick={() => fileInputRef.current?.click()}>` - Chrome 安全策略阻止
2. ❌ `<label>` 包裹 `<input type="file">` - 无反应
3. ❌ `<label htmlFor="epub-file-input">` - 无反应
4. ❌ `<label htmlFor>` + onClick 双重触发 - 无反应

**建议尝试的新方向**：
- 使用原生按钮直接触发
- 检查 CSS pointer-events 属性
- 简化弹窗结构，移除外层 onClick
- 让 input 元素可见测试
